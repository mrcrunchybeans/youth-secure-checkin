Building Custom Integrations
Integration Development Guidelines

Want to build your own integration? Here are the guidelines for creating custom integrations:

The system uses a modular Flask architecture with SQLite database. Each integration should:

  • Be implemented as a separate Python module (e.g., my_integration_client.py)
  • Have its own configuration stored in the settings table
  • Include enable/disable functionality
  • Provide clear error handling and logging
# Example integration structure
class MyIntegrationClient:
    def __init__(self, api_key, api_secret):
        self.api_key = api_key
        self.api_secret = api_secret
    
    def sync_data(self):
        # Your sync logic here
        pass

Store integration settings in the settings table:

conn = get_db()
conn.execute(
    "INSERT OR REPLACE INTO settings (key, value) VALUES (?, ?)",
    ('my_integration_enabled', 'true')
)
conn.commit()

Access the database safely:

def get_db():
    conn = sqlite3.connect('data/checkin.db')
    conn.row_factory = sqlite3.Row
    return conn

Create dedicated routes for your integration:

@app.route('/admin/my_integration')
@require_auth
def my_integration():
    # Your integration page logic
    return render_template('admin/my_integration.html')

Add your integration card to this page by editing templates/admin/integrations.html.

  • Error Handling: Always use try/except blocks and log errors appropriately
  • User Feedback: Use Flask's flash() for user notifications
  • Security: Store sensitive credentials securely, never in plain text logs
  • Testing: Provide a "Test Connection" button for users to verify setup
  • Documentation: Include clear setup instructions in your integration card
  • Scheduling: Use APScheduler for automated sync tasks
  • Rate Limiting: Respect API rate limits of external services

Here's a minimal working example:

# my_integration_client.py
import requests

class MyIntegrationClient:
    def __init__(self, api_key):
        self.api_key = api_key
        self.base_url = "https://api.example.com"
    
    def test_connection(self):
        try:
            response = requests.get(
                f"{self.base_url}/ping",
                headers={"Authorization": f"Bearer {self.api_key}"}
            )
            return response.status_code == 200
        except Exception as e:
            print(f"Connection error: {e}")
            return False

# In app.py
@app.route('/admin/my_integration', methods=['GET', 'POST'])
@require_auth
def my_integration():
    if request.method == 'POST':
        api_key = request.form.get('api_key')
        enabled = request.form.get('enabled') == 'on'
        
        conn = get_db()
        conn.execute(
            "INSERT OR REPLACE INTO settings (key, value) VALUES (?, ?)",
            ('my_integration_api_key', api_key)
        )
        conn.execute(
            "INSERT OR REPLACE INTO settings (key, value) VALUES (?, ?)",
            ('my_integration_enabled', str(enabled))
        )
        conn.commit()
        conn.close()
        
        flash('Integration settings saved!', 'success')
    
    return render_template('admin/my_integration.html')
