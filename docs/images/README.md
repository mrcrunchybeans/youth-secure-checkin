## Screenshot Guidelines

**Format**: PNG (preferred) or JPG
**Size**: 1200-1600 pixels wide (for good quality on GitHub)
**Content**: 
- Use test/demo data (no real family information)
- Show the interface in action
- Include relevant UI elements
- Ensure good lighting/contrast

**Tools**:
- Windows: Snipping Tool or Snip & Sketch
- Mac: Cmd+Shift+4
- Linux: GNOME Screenshot or Flameshot

## Adding Screenshots

1. Take screenshots of each view
2. Save with descriptive filenames (as listed above)
3. Place in this `docs/images/` directory
4. Commit and push to repository
5. Screenshots will automatically display in README

## Privacy Note

**Never include:**
- Real family names
- Real phone numbers
- Actual addresses
- Any personally identifiable information (PII)

Use test data like:
- Names: John Smith, Jane Doe, Test Family
- Phones: 555-1234, 555-0000
- Groups: TEST-001, Demo Troop
